# 🏨 Sistema Hotelero - Brisas de Mayo

## 🚀 Tecnologías
- Laravel (API)
- React + Vite
- TailwindCSS
- MySQL
- Sanctum (Auth)

## ⚙️ Instalación

### Backend
cd backend
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate
php artisan serve

### Frontend
cd frontend
npm install
npm run dev
